About JustBuild

JustBuild is a set of tools designed to enhance your Chat Assisted Programming experience. We aim to smooth out the workflow when using ChatGPT, Claude, Copilot, or other AI assistants for coding, making your development process faster and more efficient.

Features

Seamless Integration

Effortlessly integrate AI-generated code snippets into your existing projects.

Smart Merging

Intelligently merge code blocks while preserving existing functionality.

Efficient Handling

Seamlessly handle "code remains the same" comments for a smoother workflow.

Git-like Diff Viewing

Easily compare and understand code changes with familiar diff-style views.

Getting Started

Installation


npm install -g justbuild
# or
yarn global add justbuild

Basic Usage


# Paste new code into your file
lfg paste ./your_program.py

# Merge changes for the whole repo
lfg merge

# Merge changes for a specific file
lfg merge old_file.py new_code_from_llm.py
