The Communication Challenge

The fundamental issue in AI-assisted coding lies in bridging the communication gap between three distinct paradigms:

Humans: Graphs

We think in interconnected networks of ideas and visualize complex interactions.

Computers: Trees

Programs operate on hierarchical structures, like abstract syntax trees.

Language Models: Lines

AI models process and generate information sequentially as linear text.

Our Tools: Bridging the Gap

Python Library: justbuild

A powerful library with the 'lfg' command line tool to help convert between graphs, trees, and lines. It assists in merging LLM outputs back into code, planning tools, and building planning path DAGs.
